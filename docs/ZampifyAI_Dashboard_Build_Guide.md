# Zampify AI Dashboard Build Guide

## Personas

-   AP Specialist
-   Accounting Manager
-   Platform Administrator

## Shared Layout

Top Bar: Logo, Search, Notifications, Profile Sidebar: Dashboard,
Communication Center, Processing Pipeline, Review Queue, History,
Analytics, Settings

## AP Specialist

### Communication Center

Inbox, Vendor Queries, Invoices, Exceptions, Spam, Sent, Archive,
Filters, Assignment.

### Processing Pipeline

Columns: Invoice, Vendor, Amount, Stage, Risk, Priority, Assigned, SLA,
Status.

### Investigation Workspace

PDF Viewer with bounding boxes, AI Interpretation, Validation Timeline,
Action Hub.

Action Hub: Approve, Reject, Request Vendor Info, Escalate, Reprocess,
Download Audit.

## Accounting Manager

Hero KPIs: STP, Pending, Approved, Held, Avg Cycle Time. Charts: Sankey
Pipeline, Bottleneck Heatmap, Financial Snapshot, Fraud Analytics,
Vendor Performance.

## Platform Administrator

Interactive workflow graph. Nodes: Email Intake, OCR, Extraction,
Validation, Decision, Notification. Each node shows health, logs,
latency, throughput, errors.

## Shared Components

Workflow Replay, Invoice Timeline, Audit Logs, Notifications, Policy
Engine.
